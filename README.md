# ReadMe Generator #

## User Story ##
Users create many ReadMe files, therefore, they need a more streamlined way to create ReadMe's. The user requires a pre-conceived template that will help them to achieve this.

## Technologies Used ##
-NodeJS
-Javascript
-Package.json
-GitIgnore
-NPMInit
